You'll need to run a local web server to make this one work. Open up the command line and whichever command is right for your machine:

  • Python 3:python -m http.server
  • Python 2:python -m SimpleHTTPServer

Then visit http://localhost:8000/ to see your visualization. If you get an an "address already in use" error you already have a server running.
